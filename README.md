
## Memory Snake

![Memory Snake](screenshots/showcase.png "Memory Snake")

### Descrição

Uma mistura do jogo tradicional Snake com o jogo da memoria.

Desenvolvido em C com o uso da biblioteca de codigo aberto raylib (www.raylib.com).
